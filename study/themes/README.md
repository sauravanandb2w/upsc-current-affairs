# Mains theme study folders

One folder per **syllabus theme** (from `data/themes-index.json`).

```text
study/themes/<theme-id>/
├── manifest.json   # images[], sources[] (git PDFs)
├── notes.md        # eight sections (commit from app)
└── *.jpg / *.pdf
```

Open **Themes** in the CA app → pick paper → open a theme → upload cuttings / commit notes.
